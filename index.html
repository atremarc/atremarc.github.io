Tank Game

A Java based 2 player tank game featuring multiple cameras, destructible terrain, and powerups. Player 1 uses the "wasd" keys to move and the '1' key to shoot, while player 2 uses the arrow keys to move and Enter to shoot. To play, clone the repo and run csc413-tankgame-atremarc.jar.
